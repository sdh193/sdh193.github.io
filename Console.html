Help: How to open the console:
Click the inside of the bordered box. A flashing cursor should appear towards the top. If you see this you are ready to type something into the console.
Why the console may not be registering when you press the enter key:
Before pressing the enter key you must click outside of the bordered box until the flashing cursor disappears. Once it has, pressing enter will submit the expression to the system.
How the console can be used:
The console operates just like a javascript engine would. You can type in arithmetic expressions, define variables and functions, and use all the default DOM and javascript functions on the outer document and the console in your code.
How it is different from a normal console:
1. Setting a variable using "var" will not work; Instead, pretend like the variable has already been initialized before you set the value to it. For example, instead of defining and setting a variable k to 0 by typing in "var k=0", type "k=0".
2. Calling a function using "function myFunction(){}" will not work. Instead, assign the function to a variable. For example, instead of typing "function log(value){console.log(value)}" type "log=function(value){console.log(value)}"
3. Objects will not be displayed properly. If you want to see an object that is assigned to a variable properly, make sure to put it in a JSON.stringify function. Note that this will not work for html DOM objects.
4. HTML-parseable content will be auto-converted when variables containing it are submitted into the system.
5. Errors do not start with "Uncaught".
